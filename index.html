What are Hooks

  • A Hook is a special function that lets you “hook into” React features.
  • Hooks are a new addition in React 16.8.
  • They let you use state and other React features without writing a class.
  • Reuse stateful logic between components
  • Shorter, simpler way to write a react code
  • Hooks doesn't brake your old version react code. Meaning it is stricktly additive.

What are the problems hooks is trying to solve?

  • Giant components
  • Confusing components
  • Wrapper hell
  • and more...

Does hooks eliminate classes?

  • TLDR: No
  • Classes can be a large barrier to learning React. Because people need to know this in javascript.
  • Classes make code reuse and code organization more difficult
  • Classes don’t minify very well, and they make hot reloading flaky and unreliable
  • To solve these problems, Hooks let you use more of React’s features without classes.

Important: Rules of Hooks

You can use hooks and classes together (mix them in your code)

They work side by side

Lastly, use hooks only inside functional component or at the top level.

State Hook

useState is a Hook that lets you add React state to function components.

useState - It’s similar to this.setState in a class, except it doesn’t merge the old and new state together.

The only argument to useState is the initial state.

The first item is the current value, and the second is a function that lets us update it.

Example:


            const [counter, setCounter] = useState(0); // inital argument can be any type

It uses array destructuring...

Effect Hook

useState - It serves the same purpose as componentDidMount, componentDidUpdate, and componentWillUnmount in React classes, but unified into a single API.

You can do side effects here

Data fetching, setting up a subscription, and manually changing the DOM in React components are all examples of side effects

Just like with useState, you can use more than a single effect in a component

Example:


            useEffect(() => {
              // your code
              // clenaup
            }); // dependencies
            useEffect(() => {
              // your code
              // clenaup
            }); // dependencies
